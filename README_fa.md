# Slipnet Balanced Client

یک ابزار ساده دسکتاپ برای پیدا کردن DNS Resolverهای سالم برای `slipnet.exe` و سپس اجرای یک مجموعه پروکسی متعادل (balanced) به صورت محلی.

---

## این ابزار چه کار می‌کند

- یک فایل متنی شامل IPهای Resolver را می‌خواند (هر خط یک IP). خطوط تکراری به‌صورت خودکار حذف می‌شوند.  
- آن‌ها را به‌صورت همزمان (parallel) بررسی می‌کند و فقط موارد سالم را نگه می‌دارد.  
- به شما اجازه می‌دهد انتخاب کنید کدام Resolverهای سالم فعال شوند.  
- سرورهای پروکسی محلی را اجرا می‌کند:
  - SOCKS5 روی 0.0.0.0:1080
  - HTTP روی 0.0.0.0:8080
- ترافیک را از طریق Resolverهای فعال با لود بالانسینگ ساده و مدیریت خطا (cooldown) عبور می‌دهد.

---

## پیش‌نیازها

- ویندوز (چون نیاز به slipnet.exe دارد)  
- پیشنهاد: Python نسخه 3.9 یا بالاتر  
- پکیج پایتون: requests  
- فایل slipnet.exe باید در همان پوشه فایل Slipnet Balanced Client.py باشد  

نصب نیازمندی:

```bash
pip install requests
```

---

## فایل‌ها

- Slipnet Balanced Client.py → برنامه گرافیکی + منطق پروکسی  
- slipnet.exe → برنامه اصلی برای تونل کردن Resolverها  

---

## نحوه استفاده

1. اجرای برنامه:

python "Slipnet Balanced Client.py"

2. در محیط برنامه روی Browse IP List کلیک کنید و فایل لیست Resolverها را انتخاب کنید.  
3. Slipnet connection string خود را وارد کنید.  
4. مقدار Workers را تنظیم کنید و روی Start Scan کلیک کنید.  
5. Resolverهای سالم را انتخاب کنید.  
6. روی Activate Selected کلیک کنید.  
7. استفاده از پروکسی:
   - SOCKS5: your_local_ip:1080
   - HTTP: your_local_ip:8080

---

## فرمت فایل Resolver

- فایل متنی ساده  
- هر خط یک Resolver  
- خطوط خالی نادیده گرفته می‌شوند  
- خطوط تکراری حذف می‌شوند  

مثال:

1.1.1.1
8.8.8.8
9.9.9.9

---

## توضیحات رفتاری

- نتیجه اسکن شامل latency و وضعیت است  
- Resolverهای خراب حذف نمی‌شوند، فقط علامت‌گذاری می‌شوند  
- سیستم به‌صورت خودکار Resolverهای ناپایدار را موقت غیرفعال می‌کند  
- Resolverهای سالم دوباره به‌تدریج برمی‌گردند  
- هنگام بستن برنامه، همه اتصال‌ها به‌درستی بسته می‌شوند

## مزایا

- رابط گرافیکی ساده  
- اسکن سریع  
- پشتیبانی SOCKS5 و HTTP  
- پایداری بهتر  
- امکان انتخاب دستی Resolverها

## ساخت از سورس

1. به پوشه (دایرکتوری) پروژه بروید  
2. برنامه را بسازید (Build کنید)
   
```bash
pyinstaller --onefile --noconsole --name "Slipnet Balanced Client" --add-binary "slipnet.exe;." "Slipnet Balanced Client.py"
```

---

## رفع مشکل

- executable پیدا نشد → slipnet.exe کنار فایل باشد  
- Resolver سالم نیست → لیست را بررسی کنید  
- پروکسی کار نمی‌کند → پورت‌ها باز باشند  
- اسکن کند است → تعداد Worker را تنظیم کنید
